# Heredoc

[← 手順書に戻る](../3I-45-手順書.md)

**概要:**
ヒアドキュメント（heredoc）はシェルで複数行テキストを一度にファイルへ書き込む方法です。手順書の設定ファイル作成や簡易テンプレート生成に便利です。

**主なポイント:**

- `cat <<EOF > file` の形式で複数行を出力できる。
- 変数展開はデフォルトで行われ、`<<'EOF'` とすると展開を抑止できる。

**実務上の注意:**

- 機密情報の直書きは避け `.env` やシークレット管理にする。
- 権限が必要な場所へ書き込む際は `sudo` の扱いに注意する。

**例:**

```
cat <<'EOF' > .env
MYSQL_USER=wp_user
MYSQL_PASSWORD=secret
EOF
```

**参考:** シェルスクリプトのリファレンス
