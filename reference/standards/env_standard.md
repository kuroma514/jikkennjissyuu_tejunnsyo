# 環境変数（.env）ファイル

[← 手順書に戻る](../3I-45-手順書.md)

**概要:**
.env ファイルはアプリケーションや Docker Compose に渡す環境変数をまとめる簡便な方法です。機密情報の管理や環境ごとの差分を分離するのに使います。

**主なポイント:**

- `.env` を使うと Compose は自動的に変数を読み込む。
- 機密情報は可能ならシークレット管理ツールを使う。

**実務上の注意:**

- `.env` はバージョン管理から除外（`.gitignore`）し、共有時の漏洩に注意する。
- ファイルのフォーマットは `KEY=VALUE`、引用やエスケープに注意する。

**例:**

```
MYSQL_ROOT_PASSWORD=somewordpress
MYSQL_USER=wp_user
```

**参考:** Docker Compose ドキュメント（環境変数）
