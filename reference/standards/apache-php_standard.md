# Apache / PHP

[← 手順書に戻る](../3I-45-手順書.md)

**概要:**
Apache は HTTP サーバ、PHP はサーバサイドスクリプト言語です。WordPress のような PHP ベースのアプリは Apache（または Nginx）上で PHP を動かしてページを生成します（一般には LAMP 構成）。

**主なポイント:**

- Apache はリクエストを受け静的・動的コンテンツを返す。
- PHP はサーバ側で処理を行い DB と連携する。
- コンテナ環境では PHP-FPM と組み合わせる構成が一般的。

**実務上の注意:**

- `php.ini` の設定（メモリ・アップロード等）を環境に合わせて調整。
- セキュリティ: 不要関数の無効化、エラーメッセージの抑制。

**参考:** Apache・PHP の公式ドキュメント
