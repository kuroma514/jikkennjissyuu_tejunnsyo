# データベース

[← 手順書に戻る](../3I-45-手順書.md)

**概要:**
データベースはアプリケーションの永続データを管理するシステムです。RDBMS（MySQL 等）はスキーマ、トランザクション、クエリを提供し、データ整合性を保ちます。

**主なポイント:**

- データ永続化にはボリュームを使用する。
- 初期化はコンテナ起動時の環境変数で自動化できる。

**実務上の注意:**

- 定期バックアップとリストア手順を整備する。
- 認証・権限管理やネットワークアクセス制御を適切に設定する。

**コマンド例:**

```
docker exec -it <db_container> mysql -u root -p
```

**参考:** 使用している DB の公式ドキュメント
