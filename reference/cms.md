# CMS

[← 手順書に戻る](../3I-45-手順書.md)

**概要:**
CMS（Content Management System）は Web コンテンツを管理・編集するシステムで、WordPress は代表的な PHP ベースの CMS です。コンテンツ編集、テーマ運用、プラグインで機能拡張できます。

**主なポイント:**

- コンテンツ編集を非開発者でも行える仕組みを提供。
- データベースと連携して投稿や設定を永続化する。

**実務上の注意:**

- セキュリティ対策（定期アップデート、不要プラグインの削除、強固な認証）を行う。
- メディアやアップロードファイルの保存場所（ボリューム）を設計する。

**例:**

```
docker compose up -d
# ブラウザで http://localhost:8080 にアクセス
```

**参考:** 各 CMS の公式ドキュメント
