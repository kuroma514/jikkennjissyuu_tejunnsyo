# phpMyAdmin

[← 手順書に戻る](../README.md)

**概要:**
phpMyAdmin は MySQL/MariaDB をブラウザで管理するための PHP ベースの GUI ツールです。データベースの閲覧、クエリ実行、バックアップなどの日常的な運用作業を支援します。

**主なポイント:**

- ブラウザからテーブルやレコードを確認・操作できる。
- 認証情報はコンテナ起動時に環境変数で渡すことが多い。

**実務上の注意:**

- 公開環境で直接公開するのは避け、アクセス制限や認証強化を行う。
- バックアップや重要操作は CLI と組み合わせて実行する運用設計が望ましい。

**例:**

```
docker run -d -p 8888:80 --name pma phpmyadmin/phpmyadmin
```
