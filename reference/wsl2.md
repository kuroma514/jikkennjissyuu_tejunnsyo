# WSL2

[← 手順書に戻る](../3I-45-手順書.md)

**概要:**

WSL2（Windows Subsystem for Linux 2）は、Windows 上で完全な Linux カーネルを用いた軽量 VM を動かし、Linux ネイティブのツールや Docker コンテナを高い互換性で利用できる開発環境です。ローカル開発で Linux 固有の挙動を再現するのに適しています。

**主なポイント:**

- 高い互換性：フル Linux カーネルを利用するため多くのツールが動作します。
- パフォーマンス：WSL1 よりファイル I/O 性能が向上します。
- 統合：Docker Desktop と組み合わせてコンテナ実行が容易です。

**実務上の注意:**

- 開発ファイルは WSL 側（例: `/home`）に置くと I/O 性能が良い。
- Windows 側とのファイル共有ではパスや権限に注意すること。
- リソースは `%USERPROFILE%\\.wslconfig` で固定できる（大量コンテナ使用時に設定推奨）。

**コマンド例:**

```
wsl -l -v
```
